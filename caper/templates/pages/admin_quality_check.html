How to Fix Orphaned Projects

These projects are marked as old versions (current=False) but have no version chain. To fix them, use Django shell: 

# Set a project to current (make it active):
from caper.utils import collection_handle
from bson.objectid import ObjectId

collection_handle.update_one(
    {'_id': ObjectId('PROJECT_ID_HERE')},
    {'$set': {'current': True}}
)

# Or mark as deleted (hide it):
collection_handle.update_one(
    {'_id': ObjectId('PROJECT_ID_HERE')},
    {'$set': {'delete': True}}
)
